# 📝 To-Do List App

A simple and interactive To-Do List application built using HTML, CSS, and JavaScript. This project helps users manage their daily tasks efficiently with persistent local storage.

## 🚀 Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 🔍 Filter tasks:
  - All
  - Active
  - Completed
- 💾 Automatically save tasks using Local Storage
- 🔄 Restore tasks after page reload
- 📱 Responsive and user-friendly interface

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## 📖 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
4. Start adding and managing your tasks.
